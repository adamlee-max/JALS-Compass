## üîπ What is the Compass?  
The Guardian‚ÄìJALS Compass is a framework for testing whether any system ‚Äî biological, financial, ecological, technological, or social ‚Äî is **viable**.  

It works by scoring a system across five invariants:  

1. **Boundary Symmetry** ‚Äì Every system sits between two collapse edges.  
2. **Dynamic Centre** ‚Äì The further from balance, the higher the risk.  
3. **Loop‚ÄìContinuity** ‚Äì Collapse and renewal are defaults, not bugs.  
4. **Pattern Sufficiency** ‚Äì Fragments can reveal the whole.  
5. **History/Recurrence** ‚Äì Past failures shape future survival.  

The Compass calculates a **viability score (C)** between -1.0 and +1.0:  

\[
C(\pi) = 0.3E[m] + 0.3R - 0.2Pr - 0.1H - 0.1G
\]

Where:  
- **E[m]** = ability to maintain structure  
- **R** = recovery/repair capacity  
- **Pr** = risk of breaching boundaries  
- **H** = opacity of patterns (how unclear things are)  
- **G** = historical regret (unlearned lessons)
